USBprog 5.0 is a programmer / debugger (gdb) / flash tool / production tool

based on embedded linux, openocd, avrdude an python tool and a modern web gui

Features

All in one programmer and ready to use - without any installation - Windows, Mac and Linux.

Programmer

Bescause it's based on the well-known open source projects avrdude and openocd many processors are supported.

Debugger

USBprog is based on the open source jtag debugger openocd. Many ARM® processors can be debugged with GNU GDB Debugger. Choose the processor an start debugging.

Production tool

Store and recall user defined program sets. Perfect for small productions or people who will only program microcontrollers in an easy way.

Browser based

Use USBprog with your favorit browser. No installation of a special gui or a driver is needed.

Commandline tool

embeddedprog.py is a python commandline tool for windows, mac and linux with runs with no special libraries.

network connection

No additional USB driver is needed! Connect USBprog via network (USB emulated, LAN or WLAN) to your pc or tablet.

1.8V to 5.0V

Choose via software your operation target.

linux inside

USBprog 5.0 is build with embedded linux (all open source).

Updatable

Upgrade your USBprog 5.0 with our newest updates.
